# Vite + React App (scaffold)

Minimal Vite + React starter created in `vite-react-app`.

## Quick start (PowerShell)

```powershell
cd d:\Development\Mahakal-Studio-main\vite-react-app
npm install
npm run dev
```

- `npm run dev` — starts Vite dev server
- `npm run build` — builds production files to `dist`
- `npm run preview` — preview production build

Notes:
- This project uses ESM (`type: module`) and Vite. If you prefer TypeScript, I can add `tsconfig.json` and convert files.
